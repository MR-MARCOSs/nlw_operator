# devroast

O **devroast** é uma aplicação web para colar trechos de código e receber uma análise com tom de roast: direta, divertida e útil para melhorar a qualidade do código.

## O que o app oferece
- Área principal para colar código e iniciar a análise.
- Modo de roast com variação de intensidade.
- Visualização de código com destaque de sintaxe.
- Preview de leaderboard com os piores códigos ranqueados por “shame score”.
- Biblioteca de componentes de interface para evoluir a experiência com consistência.

## Estado atual
Neste momento, a interface é construída com dados estáticos para validar UX, layout e componentes antes da integração com API.

## Sobre o projeto
Este app está sendo construído durante o evento **NLW da Rocketseat**, acompanhando as aulas do evento.
